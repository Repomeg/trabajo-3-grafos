Tienes dudas?, presiona este botón


Ayuda
Primero, se deben crear ambos automatas, para esto se deben seguir los siguientes pasos:

Para crear el automata uno, primero debe ingresar el tamaño del alfabeto del automata y presionar enviar. Luego, en orden, repetir para el alfabeto de la pila y para la cantidad de conjuntos identificadores

Una vez ingresado el tamaño del alfabeto, el tamaño del alfabeto de la pila y la cantidad de conjuntos identificadores, hay que indicar los elementos de cada transacción, esto se hace seleccionando en cada casilla el valor deseado.
Una vez todos los datos estén ingresados, debe presionar el botón "enviar formulario"

Este mismo procedimiento debe repetirse en el automata 2


Una vez ambos automatas estén creados y completos, debe presionar el botón "Funciones conjuntas" para que se imprima la unión y la concatenación de los automatas

Para salir, presiona fuera de esta ventana o presiona el botón "cerrar"
